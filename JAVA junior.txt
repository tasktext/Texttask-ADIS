  Java Junior
  Требования к вакансии:

•	Опыт участия в IT проектах от 1 года
•	Знание языка программирования Java.
•	Знание языка SQL (MySQL, PostgreSQL). 
•	Опыт работы с NO SQL, Maven, ORM (MyBatis);
•	Знание Html, CSS и JavaScript, принцип CMS
•	Знание jQuery и понимание принципов работы AJAX.
•	Умение работать с SVN или GIT системой контроля версий.
•	Знание ООП.
•	Знание алгоритмических основ, структур данных, оценка сложности алгоритмов.
•	Знание основных шаблонов проектирования.
•	Знание Spring, Hibernate, JMS, JPA, OSGI, Camel
•	Опыт программирования на языках PHP, Ruby или Python.
•	Опыт работы с высоконагруженными системами
•	знание основ unix-like операционных систем (написание простых скриптов - bash, awk и т.д.);
•	опыт работы с ГИС системами (системы координат, проекции).
•	английский язык intermediate (B1)
